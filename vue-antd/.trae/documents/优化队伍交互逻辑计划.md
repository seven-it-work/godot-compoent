# 优化队伍交互逻辑计划

## 1. 交互逻辑优化

### 1.1 点击已上阵人员
- 弹出弹窗，提供选项：下阵、切换位置
- 弹窗显示队友详细信息

### 1.2 点击可上阵队友
- 弹出弹窗，提供选项：上阵
- 弹窗显示队友详细信息

### 1.3 切换位置功能
- 选择"切换位置"后，高亮所有可用的阵型格子
- 点击目标格子，实现位置交换
- 如果目标格子已有队友，进行替换
- 如果目标格子为空，直接上阵

### 1.4 上阵功能
- 选择"上阵"后，高亮所有可用的阵型格子
- 点击目标格子，将队友部署到该位置
- 如果目标格子已有队友，进行替换
