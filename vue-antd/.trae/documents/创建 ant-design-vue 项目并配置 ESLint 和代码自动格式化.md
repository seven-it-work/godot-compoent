# 创建 ant-design-vue 项目并配置 ESLint 和代码自动格式化

## 步骤 1: 创建 Vue 3 + TypeScript 项目

使用 npm create vite@latest 在当前目录创建一个基于 Vue 3 + TypeScript 的项目

## 步骤 2: 安装 ant-design-vue 依赖

安装 ant-design-vue 及其相关依赖

## 步骤 3: 配置 ESLint

1. 安装 ESLint 相关依赖
2. 初始化 ESLint 配置
3. 配置 ESLint 规则以适应 TypeScript 和 Vue 3

## 步骤 4: 配置 Prettier 代码自动格式化

1. 安装 Prettier 相关依赖
2. 创建 Prettier 配置文件
3. 配置 Prettier 规则

## 步骤 5: 配置 ESLint 与 Prettier 协同工作

1. 安装 eslint-config-prettier 和 eslint-plugin-prettier
2. 修改 ESLint 配置以集成 Prettier

## 步骤 6: 测试项目

1. 运行项目构建命令确保项目能正常构建
2. 运行 ESLint 检查确保代码规范
3. 运行 Prettier 格式化检查确保代码格式化正确

## 预期结果

- 创建一个基于 Vue 3 + TypeScript 的 ant-design-vue 项目
- 配置好标准的 ESLint 规则
- 配置好代码自动格式化功能
- 项目能正常构建和运行
