# 炉石酒馆战棋选择英雄界面设计方案

## 实现内容

### 1. 界面设计
- **宽度设置**：页面宽度设置为100%，适应不同屏幕尺寸
- **布局结构**：居中对齐的英雄选择界面，包含标题和英雄卡片列表
- **英雄卡片**：每个英雄以卡片形式呈现，包含以下信息：
  - 英雄头像（使用首字母占位符）
  - 英雄名称
  - 生命值和护甲
  - 英雄技能名称、描述和消耗

### 2. 功能实现
- **英雄数据**：内置3个英雄数据（雷诺·杰克逊、伊利丹·怒风、玛维·影歌）
- **选择功能**：点击英雄卡片可以选择英雄，显示选择结果提示
- **响应式设计**：适配移动端和桌面端

### 3. 技术细节
- 使用Vue 3 + TypeScript开发
- 采用组件化设计，便于后续扩展
- 现代化的CSS样式，包含悬停效果和动画
- 清晰的代码结构，便于维护

## 运行方式

1. 安装依赖：`npm install`
2. 启动开发服务器：`npm run dev`
3. 访问 http://localhost:5173/ 查看选择英雄界面

- **视觉吸引力**：现代化的深色主题设计，符合游戏风格
