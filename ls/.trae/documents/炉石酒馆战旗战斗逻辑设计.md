# 炉石酒馆战旗战斗场景静态页面设计

## 1. 设计思路

### 1.1 布局结构
- **竖屏设计**：整体采用竖屏布局，符合移动端使用习惯
- **左右对战**：左侧为敌方区域，右侧为玩家区域
- **7位置布局**：每侧从上到下排列7个随从位置
- **清晰区分**：通过颜色、边框等视觉元素区分敌我

### 1.2 视觉设计
- **敌方区域**：红色边框，暗红色背景，标识为"敌人"
- **玩家区域**：蓝色边框，浅蓝色背景，标识为"玩家"
- **随从位置**：每个位置为一个CardSlot组件，空位置显示虚线边框
- **战斗信息区**：底部显示战斗状态、回合数等信息
- **控制按钮区**：提供战斗控制按钮（开始战斗、跳过等）

### 1.3 交互设计
- **静态展示**：初始为静态页面，展示战斗场景布局
- **战斗中**：随从位置动态显示战斗状态
- **战斗结束**：显示战斗结果和奖励

## 3. 开发步骤

### 3.1 第一步：创建静态战斗场景组件
1. **创建BattleScene.vue组件**：实现战斗场景的HTML结构和CSS样式
2. **设计7位置布局**：每侧从上到下排列7个CardSlot组件
3. **添加视觉样式**：区分敌我区域，美化UI
4. **集成到index.vue**：添加战斗场景切换逻辑

### 3.2 第二步：静态页面测试
1. **运行项目**：查看静态战斗场景效果
2. **调整样式**：根据实际效果调整布局和样式
3. **用户确认**：用户确认静态页面满意后进行下一步

### 3.3 第三步：后续逻辑开发（待静态页面确认后）
1. **战斗逻辑开发**：实现战斗流程、攻击顺序、伤害计算等
2. **特殊效果处理**：圣盾、剧毒、亡语等效果
3. **动画效果**：添加战斗动画，提升视觉体验
4. **战斗结算**：实现战斗结果计算和奖励发放

## 4. 预期效果

### 4.1 静态页面效果
- 清晰的左右对战布局
- 每侧7个随从位置，从上到下排列
- 明显的敌我区分
- 美观的视觉设计
- 响应式布局，适配不同屏幕尺寸

### 4.2 交互效果（静态页面阶段）
- 可切换到战斗场景
- 可查看战斗布局
- 可返回主界面

## 5. 技术要点

- **Vue 3 Composition API**：使用最新的Vue技术
- **响应式设计**：适配不同屏幕尺寸
- **组件化开发**：复用CardSlot组件
- **CSS渐变**：使用渐变背景增强视觉效果
- **Flex布局**：实现灵活的页面布局
